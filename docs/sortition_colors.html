Key Insights

Color Corrections

#501D43 is actually a deep plum/purple-brown (not pure brown) - it has magenta undertones (HSL hue: 315°).

#841E40 is a burgundy with red undertones (HSL hue: 340°), distinct from the plum.

Usage Hierarchy

  1. Primary Orange (#D7764E): Call-to-action buttons, links, key highlights
  2. Secondary Plum (#501D43): Headings, logo, navigation
  3. Tertiary Burgundy (#841E40): Section backgrounds, secondary elements
  4. Neutral Light (#F7F7F7): Page backgrounds, cards
  5. Neutral Dark (Black): Alternative sections, footer

Accessibility Recommendations

  • Use orange for large text only on white (18px+ or 14px+ bold)
  • Use #501D43 or black for body text on light backgrounds
  • White text works excellently on all dark backgrounds
  • All dark colors provide excellent contrast for text
