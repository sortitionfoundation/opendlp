{{ _("Parameters") }}

assembly_id UUID ✅ Target assembly ID
raw bytes ✅ PDF bytes. Sent as pdf_base64 over the wire and decoded server-side.

{# Returns #}

{{ _("Returns") }}

RegistrationDocument → Stored document metadata (existing row on dedup)
{# Error Cases #}

{{ _("Error Cases") }}

  • DocumentValidationError — too large, or the bytes are not a PDF
  • DocumentQuotaExceeded — page already at the per-page document limit
  • RegistrationPageNotFoundError — assembly has no registration page
