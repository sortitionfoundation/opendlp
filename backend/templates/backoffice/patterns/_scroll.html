{# Overview Card #} {% call card(title="Scroll Preservation Overview", composed=true) %} {% call card_body() %} {# Description #}

{{ _("Scroll preservation prevents the jarring experience of losing your scroll position when navigating or filtering. The system saves the current scroll position in the URL as an ephemeral scroll parameter, restores it on page load, then immediately cleans the URL.") | safe }}

{# Philosophy #}

{{ _("Design Philosophy") }}

  • {{ _("Ephemeral") }}: {{ _("Scroll parameter exists only during page transition — removed immediately after restoration") }}
  • {{ _("URL-based state") }}: {{ _("Testable, shareable, bookmarkable (without scroll param in final URL)") }}
  • {{ _("Zero configuration") }}: {{ _("Global restoration runs automatically on any page with scroll param") }}
  • {{ _("CSP-safe") }}: {{ _("No inline scripts required") }}

{{ _("Alpine.js directive that automatically applies scroll preservation to all links within an element. Links can opt-out with data-no-scroll-preserve attribute.") | safe }}

{# When to Use #}

{{ _("When to Use") }}

  • {{ _("Pagination navigation (already included in pagination component)") }}
  • {{ _("Any container with multiple navigation links") }}
  • {{ _("Table rows with action links") }}
{# Code Example #}

{{ _("Code") }}

{# Container with auto-scroll preservation #}

    <a href="?page=1">1</a>  {# scroll preserved #}
    <a href="?page=2">2</a>  {# scroll preserved #}
    <a href="/dashboard" data-no-scroll-preserve>Home</a>  {# opt-out #}

{# Code Example: With Confirmation #}

{{ _("With Confirmation Dialog") }}

{# Form with confirmation - uses data-preserve-scroll attribute #}
<form method="post"
      action="{% raw %}{{ url_for('my.delete_item', id=item.id) }}{% endraw %}"
      x-data
      data-preserve-scroll
      @submit.prevent="$confirm('Are you sure?', $el)">
    <input type="hidden" name="csrf_token" value="{% raw %}{{ csrf_token() }}{% endraw %}">
    <button type="submit">Delete</button>
</form>

{# The $confirm magic checks for data-preserve-scroll and adds scroll param #}
{# Code Example: Without Confirmation #}

{{ _("Without Confirmation Dialog") }}
