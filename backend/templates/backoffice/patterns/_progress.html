⚠️ {{ _("Architecture Notes") }}

  • progress_bar() — {{ _("Stateless macro with primitive parameters (label, current, total)") }}
  • progress_indicator() — {{ _("Wrapper that accepts a ProgressInfo dataclass object") }}
  • {{ _("Server-side rendering") }} — {{ _("Progress is polled via HTMX, not WebSocket") }}
