{# Focus Preservation Overview #} {% call card(title="Focus Preservation", composed=true) %} {% call card_body() %} {# Description #}

{{ _("Preserves keyboard focus across page reloads. When a control like a dropdown triggers a page reload, the focus is automatically restored to that element after the page loads. This improves accessibility for keyboard users.") }}

{# When to Use #}

💡 {{ _("When to Use") }}

  • {{ _("Filter dropdowns") }} — {{ _("When selecting a filter value reloads the page") }}
  • {{ _("Navigation controls") }} — {{ _("Any control that navigates to the same page with different params") }}
  • {{ _("Data source selectors") }} — {{ _("Like the assembly data source dropdown") }}
{# How it Works #}

{{ _("How it Works") }}

  1. {{ _("Add data-focus-id=\"unique-id\" to the focusable element") | safe }}
  2. {{ _("When navigating, append #focus=unique-id to the URL") | safe }}
  3. {{ _("On page load, the system finds and focuses the element with that ID") }}
  4. {{ _("The URL hash is cleaned up after focus is restored") }}

{# Code Example #}

{{ _("Code") }}

<select x-data="{ selected: '' }"
        x-model="selected"
        @change="window.location.href = $focusUrl('/page?param=' + selected)"
        data-focus-id="my-select">
    <option value="a">Option A</option>
    <option value="b">Option B</option>
</select>

{# Key Points #}

⚠️ {{ _("Key Points") }}

  • data-focus-id — {{ _("Must be unique on the page") }}
  • {{ _("Only on keyboard focus") }} — {{ _("Focus hash is only added when element has keyboard focus (not mouse click)") }}
  • {{ _("URL cleanup") }} — {{ _("The #focus hash is removed from URL after focus is restored") }}
