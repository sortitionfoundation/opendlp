Accessibility Features (Built-in)

  • Combobox pattern: role="combobox" on input with aria-autocomplete="list"
  • Listbox: role="listbox" on dropdown with aria-label from label parameter
  • Options: role="option" with aria-selected indicating highlighted item
  • Live region: Screen reader announces result count (e.g., "3 results available")
  • Active descendant: aria-activedescendant tracks highlighted option

Keyboard Navigation

  • Arrow Down: Move to next option
  • Arrow Up: Move to previous option
  • Enter: Select highlighted option
  • Escape: Close dropdown
