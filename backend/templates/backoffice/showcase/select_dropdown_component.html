Built-in Accessibility

  • Native element: Uses <select> with full browser accessibility support
  • Label association: for/id linking for screen readers
  • Descriptions: aria-describedby links hint and error text
  • States: aria-required, aria-invalid, aria-disabled

Keyboard Navigation

  • Arrow Up/Down: Navigate between options
  • Enter/Space: Open dropdown or select highlighted option
  • Escape: Close dropdown without selecting
  • Type letter: Jump to first option starting with that letter
  • Home/End: Jump to first/last option
