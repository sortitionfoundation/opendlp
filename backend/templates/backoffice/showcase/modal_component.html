{% endcall %} {{ showcase_subtitle("HTMX Polling Behavior") }}

In production, the progress_modal uses HTMX for automatic updates:

  • hx-get - Server endpoint that returns updated modal HTML
  • hx-trigger="every 1s" - Polls server every 1 second while task is running
  • hx-swap="outerHTML" - Replaces entire modal with server response
  • Polling stops automatically when task completes (server omits hx-get attribute)
  • Auto-scroll keeps message log at bottom as new messages arrive
